FPLAIN.DAT

FPLAIN.DAT information goes into grid table:

**gpkg table: grid**

- "fid" INTEGER PRIMARY KEY,
- 'cell_north' INTEGER,
- 'cell_east' INTEGER,
- 'cell_south' INTEGER,
- 'cell_west' INTEGER,
- 'n_value' REAL,
- 'elevation' REAL,
- 'geom' POLYGON
